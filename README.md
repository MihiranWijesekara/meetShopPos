
# 🛒 Chicken POS System
A simple and smart Point of Sale (POS) system for managing sales and inventory in real-time.

## 📌 Features
-View daily, weekly, and monthly sales reports
-Access sales data from all previous months
-View current stock levels
-Access stock history from all previous months
-Generate monthly product reports, including:
  -Total quantity purchased (stock added)
  -Total quantity sold
  -Total profit for each product
